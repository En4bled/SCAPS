NOTA DE DESARROLLO

Este proyecto ha sido desarrollado de forma independiente por un único desarrollador, utilizando tecnologías de Inteligencia Artificial de vanguardia para la asistencia en programación y diseño.

Actualmente, el juego se encuentra en una fase muy temprana de desarrollo. El objetivo principal es experimentar con mecánicas de físicas en 2D y conectividad multijugador.

Se han tomado ideas, conceptos y mecánicas inspiradas en diversos títulos populares de la industria para crear esta experiencia. No obstante, cualquier similitud directa con productos comerciales existentes es meramente circunstancial y coincidencia absoluta.

NOTAS DE DESARROLLO

VERSIÓN 1.6.0 - GAME FEEL & PHYSICS (ACTUAL)

  • + Sistema de Post-procesado Bloom (resplandor) global y configurable.

  • + Reajuste de masas y transferencia de inercia coche-balón.
  • + Respawn inteligente (Fijo para P1, aleatorio para Bots).

VERSIÓN 1.5.0 - BANNER DESIGNER & PROFILE V2

  • + Diseñador de Banners avanzado: degradados, patrones y colores pro.
  • + Rediseño de Interfaz de Perfil en 2 columnas con vista previa 0.65x.
  • + Iconografía animada (GIFs) en navegación de pestañas.

VERSIÓN 1.4.0 - ADAPTABILITY & CONTENT

  • + UI 100% Adaptativa mediante unidades Viewport (1080p/2K/4K).

  • + Sistema de Previsualización dinámica de Boosts y Explosiones.

VERSIÓN 1.3.0 - PERSONALIZACIÓN & PRO TOOLS

  • + Persistencia USER.CONFIG en localStorage (Ajustes guardados).

VERSIÓN 1.2.0 - CICLO DE VIDA

  • + Sistema de Game Over con estadísticas detalladas y revancha.
